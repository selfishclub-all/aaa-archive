02 — Admin Flow
어드민 동작 원리
브라우저에서 GUI로 편집 → GitHub API로 커밋 → Vercel 자동 배포. 서버 없이 동작합니다.
01

🔐 인증

GitHub Personal Access Token을 입력하면 localStorage에 저장. 이후 모든 API 요청에 토큰 사용.

localStorage.setItem('gh_token', token)
02

📄 파일 목록 로드

GitHub Contents API로 해당 컬렉션 폴더의 .md 파일 목록을 가져옴. 클릭 시 개별 파일 내용 로드.

GET /repos/:owner/:repo/contents/:path
03

✏️ Frontmatter 편집

Base64 디코딩 → YAML frontmatter 파싱 → GUI 폼 렌더링. 수정 후 다시 YAML로 직렬화.

parseFrontmatter() → renderFieldHtml()
04

💾 저장 & 배포

수정된 내용을 Base64 인코딩 → PUT API로 커밋. selfishclub/aaa-admin에 push되면 Vercel이 자동 재배포.

PUT /repos/:owner/:repo/contents/:path

04 — Admin Features
어드민 핵심 기능
서버 없이 브라우저만으로 콘텐츠를 관리하는 3가지 핵심 시스템.
🏷️

태그 관리 시스템

카테고리·키워드를 노션 다중선택처럼 관리. 칩 클릭으로 선택/해제, hover 시 X로 즉시 삭제, 새 태그 입력으로 추가.

저장: localStorage
인사이트: 최대 3개 제한
한글 IME: isComposing 체크로 중복 방지

📝

마크다운 본문 에디터

분석·제안 탭에서 frontmatter + 본문을 함께 편집. 노션 스타일 툴바로 마크다운 삽입.

H2/H3: 제목 삽입
Bold: 선택 텍스트 굵게
목록/인용/코드: 원클릭 삽입
모달: max-w-3xl 확장

🖼️

이미지 업로드

파일 선택 또는 URL 입력으로 이미지 등록. GitHub API로 public/images/에 직접 업로드.

프리뷰: data URL 즉시 표시
경로: /images/gallery/ 또는 /members/
파일명: timestamp + 안전 문자 변환
