## 한줄 설명
HTML을 브라우저에서 열고 PNG 스크린샷으로 변환하는 도구

## 이런 때 쓰면 좋아요
- HTML 템플릿을 이미지로 변환할 때 (카드뉴스, 배너 등)
- 웹 스크래핑이나 E2E 테스트할 때

## 사용법
```bash
npm install playwright
npx playwright install
```
