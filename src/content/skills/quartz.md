## 한줄 설명
Obsidian vault를 정적 웹사이트로 변환하는 도구

## 이런 때 쓰면 좋아요
- 마크다운 노트를 웹사이트로 공개하고 싶을 때
- wikilink, 그래프 뷰, 검색이 있는 디지털 가든을 만들 때

## 사용법
```bash
git clone https://github.com/jackyzha0/quartz.git
npm install
npx quartz build --serve
```
