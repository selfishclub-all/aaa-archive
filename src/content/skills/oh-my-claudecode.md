## 한줄 설명
Claude Code를 위한 멀티에이전트 오케스트레이션 레이어

## 이런 때 쓰면 좋아요
- 복잡한 프로젝트를 여러 에이전트로 분업하고 싶을 때
- autopilot, ralph 같은 자동 실행 모드가 필요할 때
- 팀 단위로 에이전트를 운영하고 싶을 때

## 사용법
```bash
npx oh-my-claudecode init
```
