## 한줄 설명
Obsidian 안에서 Git commit/push/pull을 할 수 있는 플러그인

## 이런 때 쓰면 좋아요
- 터미널 없이 Obsidian에서 바로 push하고 싶을 때
- 자동 백업을 설정하고 싶을 때

## 사용법
커뮤니티 플러그인에서 "Obsidian Git" 검색 → 설치 → 활성화
